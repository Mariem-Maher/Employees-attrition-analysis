# Employees-attrition-analysis
Employees attrition analysis project

📊 Project Overview

  This project analyzes employee attrition to identify the key factors affecting employee turnover. The goal is to support HR teams in making data-driven decisions   to improve employee retention.

📁 Dataset

  1,470 employees
  
  32 columns (after cleaning)
  
  Public HR dataset

🛠 Tools Used

  Power BI
  
  Power Query
  
  DAX

  Python (EDA)

📊 Dashboard Pages

  Overview
  
  Money
  
  Job Nature
  
  Job Satisfaction
  
  Career Development

📌 Key Insights

  Young employees (18–25) have the highest attrition
  
  Low salary and low salary hike increase attrition
  
  Frequent travel and overtime contribute to attrition
  
  Sales Representatives are the most likely to leave
  
  Low job satisfaction and poor work-life balance are major factors

⚠️ Limitations

  The dataset does not include attrition dates, limiting time-based analysis and causal conclusions.

🎯 Purpose

  This project was developed as part of a data analytics learning journey and portfolio building.
